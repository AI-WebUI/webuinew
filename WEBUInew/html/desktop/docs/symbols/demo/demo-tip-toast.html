新增toast提示API

用法： 

var tip = new AIToastTip({
                message:"请选择营销案档次",
                type:"success",
                timeout:5000
                });

参数说明：

名称 类型 描述 默认值
message string（字符串） 消息信息。 null
type string 消息类型。分类为：success,error,info,question,warn,warn-little。 success
timeout number 显示时间，以毫秒计算。 2000

方法： 组件提供的可以使用的方法

方法名 参数 描述
open null 显示消息。
setContent string 重新设置消息内容。
setPosition number 重新设置显示位置。
