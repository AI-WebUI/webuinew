选择组件

使用方式 

//普通单选--例子
    var radio_normal = $("#radio-nomarl").AISelect({
    type: 'radio',
    displayField: 'channelName',
    valueField: 'channelId',
    data: [{
    channelId: '1001',
    channelName: '选项1',
    mustSelect: true
    }, {
    channelId: '1002',
    channelName: '选项2'
    }, {
    channelId: '1003',
    channelName: '选项3'
    }, {
    channelId: '1004',
    channelName: '选项4',
    disabled: true
    }],
    onSelect: function(obj) {
    //alert(JSON.stringify(obj.getSelectedIndex()));
    },
    must: false
    });

参数

参数名 类型 描述 默认值
type string 所使用选择组件的类型。包括:radio、radioBlock、checkbox、checkboxBlock null(必填项)
displayField string 需要显示的json数组中的字段名。 null
valueField string 显示的json数组中的唯一标识。 Today
data 数组或者json对象 填充的数据。 null
onSelect function 点击选择所触发的事件。 null
must boolean 是否必须选中至少一个。 false
background string 块级单选的另一种样式,如需使用第二种样式则添加该参数。 background(只有这一个值)

填充数据的参数 如要实现默认选中、必选、可选、不可选则需要在数据中添加以下参数:

  • 默认选中:defaultSelected:true
  • 必选且不可取消:mustSelect: true
  • 不可选:disabled: true
  • 可选不用添加任何东西,默认即为可选状态

提供方法

方法名 返回值 描述
getSelectedIndex 返回数组 获取选中的索引。
getSelectedValue 返回数组 返回选中的所有值(value:真实值)。
