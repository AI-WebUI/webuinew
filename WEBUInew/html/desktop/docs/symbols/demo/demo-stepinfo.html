带详细的文字描述流程条——JS生成

样式

流程条StepBar

使用js动态生成自适应的流程条.


    //html代码
    <div id="stepTest"></div>
    //js代码
    var data = [{
    "name": "已完成",
    "state": "finish",
    "desc": "这个是描述信息"
    },{
    "name": "进行中",
    "state": "process",
    "desc": "这个是描述信息"
    },{
    "name": "等待中等待中",
    "state": "",
    "desc": "这个是描述信息"
    },{
    "name": "等待中等待中",
    "state": "",
    "desc": "这个是描述信息"
    },{
    "name": "正在排队等待",
    "state": "",
    "desc": "这些文字每行最多八个"
    }];
    $("#stepTest").AIStepBar({
    data:data,
    width:900
    });

参数

参数名 类型 默认值 描述
data array数组 null 需要显示的步骤，每一步的名字、状态、描述。数组对象为一个一个的对象格式为：
{ "name": "进行中",
"state": "process",
"desc": "这个是描述信息"
}；
状态为三种：finish（完成）、process（进行中）、wait（等待中——默认）
width number 750 步骤条的总长度，根据传入的值自适应。
