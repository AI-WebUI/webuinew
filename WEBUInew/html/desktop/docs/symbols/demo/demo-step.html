流程条

流程条：通常用于一个流程较长的任务时；明确告知用户当前的位置，以及用户对这个任务的整体做到心里有数

样式

上一步 下一步 得到当前步数

面包屑流程条API

用法 使用 javascript 创建流程条（step）

        <div id="demo"></div>

                var opt = {
                "item": [{
                "text": "第一步"
                }, {
                "text": "第二步"
                }, {
                "text": "第三步"
                }],
                step: 2,
                visited: true,
                afterStep:lalala
                };
                var dd = $("#demo").AIStep(opt);

属性

名称 类型 描述
item array 流程条内容
text string 流程条文本
step number 默认进度
visited boolean 是否可以返回

事件

名称 参数 描述
afterStep function 切换流程条事件调用的方法名称

方法

名称 参数 描述
prevStep none 上一步
nextStep none 下一步
getStep none 获取当前步数
